Set Operations

All set operations output a set of unique values unless the keyword ALL is appended after.

UNION

\[ A \cup B\]

  • Distinct \[ |A| + |B|\]
  • ALL \[ |A| + |B|\]

INTERSECT

\[ A \cap B\]

  • Distinct \[ \min(|A|, |B|)\]

  • ALL \[ \sum_i\min(|a_i|,|b_i|), \qquad a \subseteq A,~ b \subseteq B\]

\[ A \backslash B\]
